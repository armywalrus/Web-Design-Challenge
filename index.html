Global Weather Visualization

A visualization dashboard website of weather data pulled from a random list of over 500 cities.

Using data pulled from the OpenWeatherMap API, and a randomly generated list of over 500 cities using the CityPy library, this website will allow you to see and compare results complied using Python, the Pandas library, and MatplotLib.
