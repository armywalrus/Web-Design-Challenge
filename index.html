Global Weather Visualizations



A visualization dashboard website of weather data.

This website displays plots showing weather data for over 500 cities. The intent of this project is to visualize weather patterns as one moves closer to the equator.

For each city, four factors were considered by their latiudes: wind speed, temperature, cloudiness, and humidity.

Project Details

This website was created using a combination of HTML, CSS, and JavaScript. HTML was used to display the data, CSS was used to format the data, and JavaScript was used for the sidebar, dropdown box on the navigation bar, and the popup windows on the visualization pages.

The weather visualizations displayed on this site were generated using Python. A randomly generated list of over 500 cities was pulled using the CityPy library. The data for those specific cities was then pulled from the OpenWeatherMap API. After the data was gathered, it was manipulated using the Pandas library, in order to generate the visualizations shown on this site using matplotlib.
