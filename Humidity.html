The purpose of this project was to pull weather data from the OpenWeather API in order to generate plots that show how weather patterns change as proximity to the equator increases. Four factors were examined: Wind Speed, Temperature, Cloudiness, and Humidity. This shows the comparison of latitude versus humidity. As the visualizations show, humidity shows a visible increase as you move closer to the equator.

This visualization shows a distinct increase in humidity as one gets farther away from the equator.
